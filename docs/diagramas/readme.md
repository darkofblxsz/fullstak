# Diagramas de Arquitectura

En esta carpeta se deben almacenar los diagramas correspondientes a la arquitectura del sistema.

## Diagramas Sugeridos
1. **Diagrama de Contenedores (C4 Model)**: Mostrando cómo interactúan el API Gateway, el Eureka Server, las bases de datos y los microservicios individuales.
2. **Diagrama de Secuencia de Compra**: Ilustrando el flujo desde que el cliente añade productos al carrito, procesa el pedido, efectúa el pago, se emite la boleta y se programa el envío.
3. **Diagrama Entidad-Relación**: Mostrando el modelo relacional unificado detallado en `bd-general.sql`.

## Formato
Se recomienda utilizar formatos de imagen estándar (`.png`, `.svg`) o código de diagramas como `.drawio` o Mermaid `.md`.
