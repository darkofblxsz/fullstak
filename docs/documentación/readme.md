# Documentación General del Ecosistema

Esta carpeta contiene la documentación operativa, guías de instalación y manuales de administración del ecosistema de microservicios.

## Contenido Recomendado
1. **Guía de Despliegue Local**: Pasos para arrancar los servicios en orden (Eureka -> Config/Gateway -> Microservicios).
2. **Guía de Configuración de Entorno**: Variables necesarias para conectarse a las bases de datos MySQL locales o de staging.
3. **Manual de Pruebas**: Instrucciones para ejecutar las pruebas unitarias y de integración de todo el proyecto.
